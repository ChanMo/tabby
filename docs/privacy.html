Tabby 隐私政策

Tabby 是一个 Chrome 插件，使用人工智能为您提供文本补全建议。

隐私

Tabby 是一个本地运行的 Chrome 插件。这意味着 Tabby 不会收集或存储任何用户数据，包括您的 IP 地址、浏览历史记录或任何其他个人身份信息。

Tabby 使用的文本补全接口取决于您在插件设置中提供的地址。这些接口可能会收集或存储用户数据，但 Tabby 对此不负责。

用户数据

Tabby 不收集或存储任何用户数据。这包括您的 IP 地址、浏览历史记录或任何其他个人身份信息。

文本补全接口

Tabby 使用的文本补全接口取决于您在插件设置中提供的地址。这些接口可能会收集或存储用户数据。

隐私政策的变更

Tabby 保留随时更新本隐私政策的权利。如果我们对本隐私政策进行任何重大更改，我们将在我们的网站上发布通知。
